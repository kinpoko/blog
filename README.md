## Usage
This needs MySQL ,and this is not included in commom.php. So you need create it.
Create a database named `blog`:

``` 
CREATE TABLE post (
  id SERIAL,
  url TEXT,
  title TEXT,
  description TEXT,
  tag TEXT,
  content TEXT,
  datestamp DATE
);
```

and `commom.php`:
```PHP:commom.php
<?php
$dsn = 'mysql:dbname=***;host=***';
$user = '***';
$password = '***';

$pdo = new PDO($dsn, $user, $password);
?>
```
